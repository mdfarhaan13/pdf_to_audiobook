# pdf_to_audiobook
A Python app that converts PDFs to audio using gTTS.

## Features
- Extract text from PDFs
- Convert text to speech (gTTS)
- Save as MP3 file
- Simple Tkinter GUI

## Requirements
- Python 3.x
- PyPDF2 (`pip install PyPDF2`)
- gTTS (`pip install gTTS`)
